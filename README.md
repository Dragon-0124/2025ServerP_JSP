# 2025 Server-Side Programming & JSP Archive

This repository serves as a technical backup for the "Server Programming" course taken in 2025. It focuses on building robust backend systems using Java-based web technologies.

## 🚀 Learning Objectives & Core Concepts

### 1. Java Server Pages (JSP) & Servlets
- **Lifecycle Management**: Understood the initialization, service, and destruction phases of Servlets and JSPs.
- **MVC Architecture**: Practiced separating concerns by using Servlets as Controllers, JSPs as Views, and Java Beans as Models.
- **Implicit Objects**: Utilized `request`, `response`, `session`, and `application` objects to manage data flow and user state.

### 2. Backend Logic & Data Handling
- **Database Connectivity (JDBC)**: Implemented CRUD (Create, Read, Update, Delete) operations by connecting web applications to relational databases (MySQL/Oracle).
- **Connection Pooling**: Learned to optimize database performance using Connection Pools (DBCP).
- **Business Logic**: Developed server-side algorithms to process user input and return dynamic results.

### 3. Web Server Configuration
- **Apache Tomcat**: Configured and deployed web applications on the Tomcat servlet container.
- **Deployment Descriptor (web.xml)**: Managed servlet mapping and filter configurations.
- **Session & Cookie Management**: Implemented secure user authentication and persistent login features.

## 📂 Project Structure
- `/src`: Java source files (Servlets, DAOs, DTOs).
- `/WebContent`: JSP files, HTML, CSS, and client-side assets.
- `/lib`: External libraries and JDBC drivers.
